# markdownからPowerPointのスライドを作る奴

私からの指示はこのファイルに書きます、あなたの応答もこのファイルにまとめて書いて下さい。

## STEP1

### 私からの指示

内容のMarkdownファイルからパワーポイントのスライドを作る仕組みを作ってほしいです。
まずは何の言語で作るのが良いか考えてください。

### 回答

Pythonが最も適しています。理由は以下です。

- `python-pptx` という成熟したライブラリでPPTX生成ができ、機能と安定性のバランスが良い
- Markdown解析は `markdown` / `mistune` などの定番ライブラリがあり実装が簡潔になる
- この環境にPythonがある前提なので、導入コストが低い
- CLI化しやすく、将来の拡張（テンプレート対応やレイアウト指定）もやりやすい
